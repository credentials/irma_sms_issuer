Adding a mobile phone number attribute

On this page you can add a (mobile, Dutch) telephone number as a new attribute to your IRMA app. This can be done multiple times, for different numbers. It works via the following three steps.

  1. Fill in your phone number below, and click the 'Send SMS' button.
  2. A text message will then be sent to this number with a verfication code. Enter this code below.
  3. Scan the resulting QR-code with your IRMA-app, and follow the instructions in the app.
